## 一、项目定位

一句话：**一个能通过 AI 对话了解学生学习特点，然后自动生成定制化学习内容和路径的教育平台。**

目标课程：机器学习（面向大三计算机系学生）

## 四、架构总览

```
用户 → Web 前端 (Vue 3)
         ↓ API (HTTP/SSE)
       FastAPI 后端
         ↓
   Orchestrator（编排器）
         ↓ 意图分流（5种意图）
   ┌──────┼──────┬──────────┬──────────┐
   │      │      │          │          │
画像Agent  Tutor  资源Agent群  路径Agent  评估Agent
   │      │      │          │          │
   └──────┴──────┼──────────┴──────────┘
                 ↓
           DeepSeek V4 Flash
                 ↓
            MySQL 数据库
```

### 完整用户流程

```
用户注册/登录
    ↓
在对话页面发消息 → SSE 流式连接
    ↓
Orchestrator 判断意图（5种）
    ↓
├─ 自我介绍、普通聊天 → ProfileAgent（画像分析）
│    ├ 提取 6 维度画像 → 存 student_profiles 表
│    └ SSE 推送：chunk（聊天内容）+ profile_updated + done
│
├─ 知识问题 → TutorAgent（智能辅导）★ 新增
│    ├ 问题分类（概念/代码/应用/计算）
│    ├ 基于画像的个性化讲解
│    ├ 记录学习行为到 learning_records 表
│    └ SSE 推送：chunk（流式答案）+ tutor_analysis + done
│
├─ 生成资源 → ResourceAgents（5 个并发生成）
│    ├ 文档/题目/思维导图/代码/视频
│    ├ 存 resources 表
│    └ SSE 推送：agent_status（实时进度）+ resource_generated + done
│
├─ 规划路线 → PathAgent
│    ├ 生成带步骤的学习路径
│    ├ 存 learning_paths 表
│    └ SSE 推送：path_generated + done
│
└─ 评估学习 → EvaluationAgent ★ 新增
     ├ 分析学习记录 + 画像
     ├ 生成评估报告（优势/薄弱/建议/下一步）
     └ SSE 推送：chunk + evaluation_report + done
```

## 六、最终做成什么样

一个学生打开网站，注册登录后看到一个**有科技感但不花哨**的对话界面。

1. **第一次使用**：他会收到 AI 助手的主动问好，AI 通过自然聊天了解他的专业背景、学习目标、偏好
2. **画像素描**：AI 在对话中逐步构建他的 6 维学习画像（知识基础、认知风格、学习能力、易错点、目标、偏好），画像会在学习画像页面实时可视化
3. **资源生成**：当他说"帮我生成决策树的讲解文档"，AI 并行生成 5 种资源（文档、题目、思维导图、代码、视频脚本），整个过程通过 SSE 实时展示进度
4. **路径规划**：当他说"帮我规划学习路线"，AI 根据画像生成个性化的学习路径，按知识依赖排序
5. **持续学习**：他边学边问，AI 辅导他不懂的知识点。随着对话增多，画像越来越准确，推荐越来越个性化
6. **效果评估**：系统定期评估他的掌握程度，动态调整学习策略

最终交付物：一个**可演示、可运行**的全栈 Web 应用，后端 7 个 Agent 协作，前端可视化展示画像、资源、路径，支持流式对话。

### 2026-06-08 — 辅导Agent + 评估Agent 实现

**新增文件**：
- `app/agents/tutor_agent.py` — 智能辅导 Agent
  - 问题自动分类（概念理解/代码实现/应用场景/计算推导）
  - 基于画像的个性化讲解（因材施教策略）

  - 4 维度评估（知识掌握度/学习效率/能力成长/学习行为）
  - LLM 驱动的评估报告生成（优势/薄弱/建议/下一步）

- `app/models/knowledge_document.py` — 知识文档数据模型（knowledge_documents 表）

  - `POST /api/upload` — 上传文件并自动解析
