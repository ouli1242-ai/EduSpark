## 10.1 项目初始化（0.5天）

- [ ] 选择框架：Vue3 + Element Plus（推荐）
- [ ] 初始化项目：
  ```bash
  npm create vite@latest eduspark-web -- --template vue
  cd eduspark-web
  npm install element-plus axios
  ```

- [ ] 配置项目结构：
  ```
  src/
  ├── api/          # API调用
  ├── components/   # 组件
  ├── views/        # 页面
  ├── stores/       # 状态管理
  ├── router/       # 路由
  └── utils/        # 工具函数
  ```
