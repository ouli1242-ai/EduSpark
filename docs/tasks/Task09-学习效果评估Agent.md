- [ ] 知识掌握度：
  - [ ] 已掌握知识点数量
  - [ ] 测试正确率

- [ ] 学习效率：

  - [ ] 知识吸收速度
  - [ ] 复习频率

- [ ] 能力成长：
  - [ ] 难度等级提升
  - [ ] 解题速度变化
  - [ ] 错误率变化

- [ ] 学习行为：
  - [ ] 学习时长
  - [ ] 学习频率
  - [ ] 资源使用情况
