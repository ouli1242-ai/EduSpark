# Task 2：课程知识库构建（3天）

> 课程资料收集、文档解析、分块切割、向量化、RAG检索验证

---

## 2.1 课程资料收集

- [ ] 确定目标课程：《机器学习》（推荐）
- [ ] 收集资料来源：
  - [ ] 教材PDF（如周志华《机器学习》）
  - [ ] 课程PPT（可从学校资源或网络获取）
  - [ ] 往年习题集
  - [ ] 实验手册与代码示例
  - [ ] 学术论文（相关知识点）
- [ ] 资料整理与分类

## 2.2 文档解析

- [ ] 安装解析工具：
  ```bash
  pip install pymupdf pdfplumber python-pptx
  ```
- [ ] PDF解析：
  - [ ] 使用pymupdf（即fitz）解析，中文支持更稳定
  - [ ] 提取文本内容
  - [ ] 保留章节结构
  - [ ] 提取图片描述（如有）
- [ ] PPT解析：
  - [ ] 提取幻灯片内容
  - [ ] 保留层级关系
- [ ] 其他格式解析（Word、Markdown等）

## 2.3 内容分块切割

- [ ] 设计分块策略：
  - [ ] 按章节分块（一级分块）
  - [ ] 按知识点分块（二级分块）
  - [ ] 重叠窗口（保留上下文）
- [ ] 分块参数：
  - [ ] 块大小：500-1000 tokens
  - [ ] 重叠：50-100 tokens
- [ ] 元数据标注：
  - [ ] 章节来源
  - [ ] 知识点标签
  - [ ] 难度级别

## 2.4 文本向量化

- [ ] 选择Embedding模型：
  - [ ] 讯飞星火Embedding API（首选）
  - [ ] 或开源模型（如text2vec-base-chinese）
- [ ] 批量向量化处理
- [ ] 向量维度与归一化

## 2.5 向量数据库存储

- [ ] 安装Chroma：`pip install chromadb`
- [ ] 创建知识库Collection
- [ ] 存储向量与元数据
- [ ] 建立索引

## 2.6 RAG检索验证

- [ ] 设计测试查询（20个典型问题）
- [ ] 测试检索效果：
  - [ ] 相关性评分
  - [ ] 响应时间
  - [ ] 召回率
- [ ] 优化分块策略（如效果不佳）

---

## 输出物

- [ ] 课程知识库（Chroma数据库）
- [ ] 知识库内容清单（章节、知识点列表）
- [ ] RAG检索测试报告
- [ ] 知识库构建文档（流程、参数、注意事项）
