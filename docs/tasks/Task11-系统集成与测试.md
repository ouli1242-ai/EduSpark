## 11.4 Bug 修复记录

| 问题 | 根因 | 修复方案 |
|------|------|----------|
| 对话列表序号溢出气泡 | CSS 缺少 ul/ol/li 样式 | 新增 padding-left + list-style-position |
| 学习路径不保存到界面 | session_id 传递缺失 | 后端修复 session_id + 前端 usePathRefresh |
| 学习能力显示 NaN% | 数值字段含 NaN/Inf | 后端过滤 + 前端 Number.isFinite（4层防护） |
| 路径生成请求失败 | _handle_path 无异常捕获 | try/except + SSE 错误推送 |
| 资源生成前后端状态不一致 | SSE 未中断 | new AbortController() |
| 文档生成 str vs float | learning_ability 存储了字符串 | _safe_float() 兼容转换 |
