- [ ] 设计问题类型：
  - [ ] 概念理解类
  - [ ] 计算推导类

  - [ ] 输出问题类型

- [ ] 实现上下文关联：
  - [ ] 关联当前学习内容
  - [ ] 关联学习画像

## 8.2 文字解答生成

- [ ] 设计解答Prompt：
  ```
  学生问题：{question}
  问题类型：{type}
  相关知识点：{knowledge_points}
  学生画像：{profile}

  请生成详细解答，要求：
  - 语言适合学生水平
  - 包含步骤分解
  - 包含关键点解释
  ```

- [ ] 实现RAG检索：
  - [ ] 检索知识库相关内容
  - [ ] 注入上下文

- [ ] 实现解答生成：
  - [ ] 调用星火API
  - [ ] 流式输出
  - [ ] Markdown格式

## 8.3 图解说明生成

- [ ] 设计图像生成Prompt：
  - [ ] 从解答中提取可视化需求
  - [ ] 生成图像描述

- [ ] 调用讯飞图像生成API：
  - [ ] 生成配图
  - [ ] 保存到OSS（开发阶段用本地文件系统）

- [ ] 图文整合：
  - [ ] 文字与图像关联
  - [ ] 卡片化展示

## 8.4 语音讲解生成

- [ ] 提取讲解文本：
  - [ ] 从解答中提取关键内容
  - [ ] 转换为口语化表达

- [ ] 调用讯飞TTS API：
  - [ ] 生成语音文件

- [ ] 音文关联：
  - [ ] 语音与文字同步
  - [ ] 支持播放控制

## 8.5 多模态答案融合
